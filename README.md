## Getting Started

### Prerequisites

-   Node.js (v18+ recommended)
-   npm or yarn

### Installation

```bash
git clone https://github.com/MekuHQ/convertly.git
cd convertly
npm install
# or
yarn install
```

### Running Locally

```bash
npm run dev
# or
yarn dev
```

The app will be available at `http://localhost:5173`.

### Building for Production

```bash
npm run build
# or
yarn build
```
